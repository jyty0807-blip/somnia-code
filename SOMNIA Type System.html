Pairing Rules

EN + KR 한 문장

--somnia-font를 쓰면 영문·숫자는 Outfit, 한글은 자동으로 Pretendard로 렌더됩니다. 섞어 써도 골격이 어울립니다.

한글은 한 단계 가볍게

같은 크기에서 한글이 더 무겁게 보입니다. 본문에서 영문이 400이면 한글 장문은 300–400으로 낮춰 균형을 맞춥니다.

오버라인은 영문 대문자

Eyebrow/Overline은 Outfit 대문자 + 0.28em 자간 + 오션블루. 한글 오버라인은 자간을 0.18em로 줄입니다.

큰 글자엔 음수 자간

Display·H1은 --ls-tight(-0.02em)로 조여 단단하게, 본문은 기본 자간 + 1.7 행간으로 여유롭게.
